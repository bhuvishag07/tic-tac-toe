🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and Vanilla JavaScript.
This project demonstrates core JavaScript concepts like DOM manipulation, event handling, game logic, and conditional rendering.

🚀 Features

❌⭕ Two-player mode (Player X vs Player O)

🏆 Automatic win detection

🤝 Draw detection

🔄 Restart game functionality

🎨 Clean and responsive UI

⚡ Built with pure JavaScript (no libraries)

🛠️ Tech Stack

HTML5

CSS3

JavaScript (Vanilla JS)

🎮 How to Play

The game starts with Player X.

Players take turns clicking empty cells.

First player to align three marks horizontally, vertically, or diagonally wins.

If all cells are filled without a winner, the game ends in a draw.

Click the Restart button to play again.

🧠 Game Logic Overview

The board consists of 9 interactive cells.

Click events update the current player's mark.

The system checks predefined winning patterns after every move.

The game stops once a winner is found or a draw occurs.

Restart resets the board and game state.
